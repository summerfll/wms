# uwb_qt

## 基本介绍：

基于Qt、MySQL数据库以及UWB定位技术开发了仓储管理系统雏形，在传统仓储系统的基础上引入物联网定位技术，实现对仓库中的物料、人员实时查找与可视化追踪。

主要功能：
-------出库管理
-------入库管理
-------库存盘点
-------仓储管理
-------产品管理
-------动态盘点
-------轨迹跟踪
-------位置查找
-------视频监控
